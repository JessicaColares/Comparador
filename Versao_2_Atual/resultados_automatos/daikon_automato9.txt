Resumo Daikon:
  - 7 execucoes instrumentadas
  - Valores capturados em pontos do programa
  - Dados podem ser analisados para encontrar invariantes
  - Exemplos de invariantes que poderiam ser detectados:
     1. resultado sempre e 0 ou 1
     2. se a <= b, resultado = 0
     3. durante loop: a sempre decrementa
  - Classificacao: METODO DINAMICO

Para uso real do Daikon:
  1. Compile com instrumentacao: java daikon.DynComp test_daikon
  2. Execute: java daikon.Chicory --dtrace-file=test.dtrace ./test_daikon
  3. Analise: java daikon.Daikon test.dtrace
