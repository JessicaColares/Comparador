### executar8.c - Operações com Módulo

**Objetivo:** Testar operações com resto de divisão (módulo) e comportamento alternado entre números pares e ímpares. O loop incrementa de forma diferente dependendo da paridade do valor.

### executar9.c - Decréscimo Controlado

**Objetivo:** Demonstrar um loop que decrementa em vez de incrementar, garantindo que sempre termina. Útil para testar condições de contorno com a > b.

### executar10.c - Alternância de Operadores

**Objetivo:** Testar alternância entre soma e multiplicação a cada iteração, usando um contador de passos para decidir a operação.

### executar11.c - Condição com Limite Superior

**Objetivo:** Incluir um limitador de iterações (max_iter) para evitar loops infinitos, mostrando como proteger o código contra execuções excessivas.

### executar12.c - Comparação com Valor Absoluto

**Objetivo:** Utilizar a função abs() para comparar valores absolutos, testando diferentes ramos condicionais baseados no sinal do número.

### executar13.c - Inversão de Sinal

**Objetivo:** Demonstrar inversão de sinal como estratégia para modificar o comportamento do loop e eventualmente sair dele.

### executar14.c - Multiplicação e Divisão Alternada

**Objetivo:** Alternar entre multiplicação e divisão a cada iteração, exigindo cuidado para evitar loops infinitos quando a divisão resulta em zero.
